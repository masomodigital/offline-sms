Sending SMS Offline

Masomo SMS allows you to send messages offline in three different ways using the keyword Masomo (case-insensitive: masomo, Masomo, or MASOMO).

1. Send SMS to One Phone Number

  • Format: Masomo [Your Message] [Recipient Phone Number]
  • Example: Masomo I am coming home. Daniel 0715399449
  • Explanation: The message I am coming home. Daniel will be sent to 0715399449.

2. Reverse SMS (Send to Another Masomo User)

  • Format: Masomo [Your Message] [Recipient User ID]
  • Example: Masomo I am coming home. Daniel 23406633
  • Explanation: The message is delivered to user with ID 23406633. Sender is NOT charged.

3. Online SMS (Send to User‚Äôs Promo Code Inbox)

  • Format: Masomo [Your Message] [Recipient Promo Code]
  • Example: Masomo I am coming home. Daniel 452755
  • Explanation: The message goes to the inbox of user with promo code 452755. No airtime required!

üí° Tip

Always start your message with the keyword Masomo, followed by your content and the correct recipient identifier (phone number, user ID, or promo code).
